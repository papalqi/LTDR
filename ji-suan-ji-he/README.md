# 计算几何

### 组成

* 在计算几何当中，会用到大量的离散数学，线性代数，和其他自身的常见知识体系。
* 凸包问题，是计算几何的重中之重，因为之后的所有都会使用到这个重要的概念
* 几何求交，这其实非常的常见并且会经常运用的知识体系
* 三角剖分，计算几何的核心
* Voronoi，计算几何的重要概念和思想
* Delaunay三角剖分
* 点位置
* 几何范围搜索
* 窗口查询
